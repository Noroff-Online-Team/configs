# `@noroff/prettier-config`

Collection of Prettier configurations.

## Installation

```bash
npm install -D @noroff/prettier-config
```

## Usage

Using `.prettierrc.js`

```js
module.exports = {
  extends: ["@noroff/prettier-config"]
}
```

Using `.prettierrc.json`

```json
{
  "extends": ["@noroff/prettier-config"]
}
```
